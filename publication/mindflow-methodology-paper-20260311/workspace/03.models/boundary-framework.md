# Boundary Framework

## 必须坚定表达的内容

1. MindFlow 不是普通知识库，也不是单纯多智能体编排器
2. `Soul` 是高层约束层
3. `Execution Control` 不属于 `Mind` 七层本体，而属于 `Execution` 内部机制
4. `Inference` 是受约束推衍机制
5. 软件工程与知识工程之间存在直接关联

## 必须谨慎表达的内容

1. AI 软件将替代传统软件执行层
2. 数字人格将成为未来软件主流形态
3. 知识可执行会全面重构软件产业

## 建议措辞方式

- 用“意味着一种可能的新边界”“提示一种结构性转向”“为重新理解软件形态提供线索”
- 少用“必然”“注定”“彻底替代”这类封闭性表述

## 不宜犯的错误

1. 把趋势判断写成事实
2. 把工程蓝图直接当作理论证明
3. 把 `Mind` 写成玄学概念而非形式化结构
4. 把论文写成产品阐释或研究报告
