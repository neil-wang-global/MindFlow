# MindFlow 方法论论文项目

本项目使用 POMASA 组织一套面向学术论文生产的多智能体流程，目标是产出一篇面向中文学术期刊的中文方法论论文，研究对象为 MindFlow 所体现的 AI-native 方法论与知识管理一体化框架。

## 目录结构

- `user_input_template.md`：用户输入与研究约束
- `agents/`：论文生产 Agent Blueprint
- `references/domain/`：核心概念与术语边界
- `references/methodology/`：研究方法、数据来源、写作模板、质量标准
- `workspace/`：中间资料、理论建模结果、图表方案与论文草稿
- `_output/`：导出产物
- `scripts/`：DOCX / PDF 导出脚本
- `wip/`：过程笔记

## 研究目标

1. 提出 `Mind` 方法论框架及其理论命题
2. 论证 MindFlow 与知识管理、能力内化、组织目标约束之间的关系
3. 讨论“软件工程是知识工程”与“知识可执行”的理论意义
4. 通过案例与范式对比增强论文的可发表性
5. 形成一份具备投稿结构完整性的论文草稿
