# Data Sources Guide

## Data Source Types

### 1. 项目一手资料

- Description: MindFlow 本地仓库与 GitHub 仓库中的正式文档、目录结构、模板与协议
- Priority: High
- Credibility: High
- Examples:
  - `README-CN.md`
  - `SYSTEM.md`
  - `mind/` 目录各层说明

### 2. 学术论文与正式出版资料

- Description: 与 MAS、知识管理、知识工程、认知架构、组织学习、AI 软件相关的论文、综述、正式出版文献
- Priority: High
- Credibility: High

### 3. 官方技术文档与原始工程资料

- Description: 相关框架、论文作者主页、官方文档、研究组织原始说明
- Priority: High
- Credibility: Medium-High to High

### 4. 高质量行业与技术分析

- Description: 用于补充框架讨论与现实背景，但不应替代学术来源
- Priority: Medium
- Credibility: Medium

## Sources To Use Cautiously

- 仅有二手转述、缺乏一手出处的文章
- 观点化博客而无明确事实支撑者
- 无法确认作者与时间的非正式内容

## Recording Format

每条原始资料记录至少包括：

- 标题
- 来源类型
- URL 或文献信息
- 访问日期
- 发布时间
- 对应研究问题
- 核心内容摘要
- 关键原句
- 可信度判断
- 备注
